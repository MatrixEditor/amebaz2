Expand description

Represents a firmware partition record.

A Record struct encapsulates information about a partition within a firmware image. This record stores a hash key for the partition, which will be later used to verify the signatures of the partition contents.

Fields§

§start_addr: u32

The starting address of the partition in the firmware image (4 bytes).

§length: u32

The length of the partition in bytes (4 bytes).

§part_type: PartitionType

The partition type (1 byte). This is an enum (PartitionType).

§dbg_skip: bool

A flag that indicates whether debugging should be skipped for this partition (1 byte). true means skip debugging, false means do not skip.
